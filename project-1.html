Project Overview

This was the introductory project during my journey through the full-stack development bootcamp at SoyHenry. In this project, I was able to apply different types of designs, as well as implement the functionality to add a card that can also be deleted. It is a project that only has a frontend; therefore, when the page is refreshed, any created cards are deleted.
