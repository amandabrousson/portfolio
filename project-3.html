Project Overview

A full-stack web application with a Single Page Application (SPA) structure. It includes an HTTP server for managing appointments from a database.

The application is designed for a kinesiology clinic, allowing users to book their appointments online. To make a reservation, users must be authenticated. Appointments can only be booked within the clinic's working hours and on business days. Appointments can be canceled up to 24 hours in advance but cannot be rescheduled.
