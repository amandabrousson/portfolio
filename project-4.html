Project Overview

Full-stack web application that consists of a pre-existing backend and a newly developed frontend. The challenge was to build a Single Page Application (SPA) for an e-commerce platform based on the provided backend code. The frontend was developed using Next.js and Tailwind, seamlessly integrating with the backend API to provide a smooth and dynamic user experience. Key features implemented include user authentication, product navigation, shopping cart functionality, and a checkout process. I also focused on optimizing the application's performance and ensuring cross-browser compatibility.
